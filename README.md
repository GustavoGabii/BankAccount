# BankAccount - Projeto Java

Projeto simples de conta bancária em Java, desenvolvido para praticar conceitos de **Orientação a Objetos**.

## Conceitos praticados
- Classes e objetos
- Atributos `private`
- Construtores
- Uso do `this`
- Métodos
- Entrada de dados com `Scanner`

## Funcionalidades
- Criar conta com nome e saldo inicial
- Depositar valor
- Sacar valor (com validação de saldo)
- Consultar saldo

## Tecnologias
- Java 21
